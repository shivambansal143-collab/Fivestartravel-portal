# FiveStarTravel.in - Premium B2B Travel Portal

## 🎉 Project Successfully Created!

A complete, premium B2B travel portal website has been built based on the FiveStarTravel.in brand.

## 📍 Project Location
```
/vercel/sandbox/fivestar-travel/
```

## 🌐 Access the Website

The development server is running at:
```
http://localhost:3000
```

## 📄 Available Pages

1. **Home** (`/`) - Landing page with hero section, features, services overview, and statistics
2. **Services** (`/services`) - Detailed B2B service offerings (Flights, Hotels, Tours, Visa, Corporate, MICE)
3. **Destinations** (`/destinations`) - International and domestic destination showcases with packages
4. **About** (`/about`) - Company information, mission, values, and team
5. **Contact** (`/contact`) - Contact form with business information
6. **Login** (`/login`) - Partner login portal
7. **Register** (`/register`) - Comprehensive partner registration form

## 🎨 Design Features

- **Brand Colors**: Navy Blue (#1e3a8a) and Orange (#f97316)
- **Custom Logo**: SVG-based star and airplane design
- **Responsive Design**: Mobile, tablet, and desktop optimized
- **Modern UI**: Gradients, shadows, hover effects, and smooth transitions
- **Professional Layout**: Clean navigation, footer with social links

## 🛠️ Technology Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS with custom color palette
- **Components**: Reusable Navbar and Footer
- **Forms**: Interactive contact and registration forms

## 🚀 Quick Start Commands

```bash
# Navigate to project
cd fivestar-travel

# Install dependencies (already done)
npm install

# Start development server (already running)
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
