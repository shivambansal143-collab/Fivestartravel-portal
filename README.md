# Fivestartravel-portal

B2B travel portal (Flight + Hotel + Visa + Package + Insurance + Tours + Transfers + Cruise + Railway + CRM + Wallet + Payments)

## Quick Start

```bash
cd fivestar-travel
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

See [fivestar-travel/README.md](./fivestar-travel/README.md) for full documentation.
